# Keywords

## Keywords

| Keyword | Description |
|---------|-------------|
| `let` | Declare a variable |
| `exit` | Exit a program with given return code |
| `print` | Print int32 to the console *(not implemented)* |
| `if/else` | Branching *(not implemented)* |
| `while` | Loop *(not implemented)* |
| `=` | Assign *(not implemented)* |

## Math operators

| Operator | Description |
|----------|-------------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `//` | Integer division |
| `%` | Modulo |

## Comparison operators *(not implemented)*

| Operator | Description |
|----------|-------------|
| `>` | Greater |
| `<` | Less |
| `>=` | Greater or equals |
| `<=` | Less or equals |
| `==` | Equals |
| `!=` | Not equals |

## Misc

| Symbol | Description |
|--------|-------------|
| `;` | End of the command |
| `{` | Beginning of a scope *(not implemented)* |
| `}` | Ending of a scope *(not implemented)* |
