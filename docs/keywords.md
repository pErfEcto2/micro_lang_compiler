# Keywords

## Keywords

| Keyword | Description |
|---------|-------------|
| `int64` | Declare an int64 variable |
| `exit` | Exit a program with given return code |
| `print` | Print int64 to the console |
| `if/else` | Conditional branching |
| `while` | Loop |
| `=` | Assign |
| `const` | Make a variable constant |

## Math operators

| Operator | Description |
|----------|-------------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `//` | Integer division |
| `%` | Modulo |

## Comparison operators

| Operator | Description |
|----------|-------------|
| `>` | Greater |
| `<` | Less |
| `>=` | Greater or equals |
| `<=` | Less or equals |
| `==` | Equals |
| `!=` | Not equals |

## Misc

| Symbol | Description |
|--------|-------------|
| `;` | End of the command |
| `(` | Opening parenthesis (used in `if` conditions) |
| `)` | Closing parenthesis |
| `{` | Beginning of a scope / block |
| `}` | End of a scope / block |
| `# ...` | Single-line comment |
| `/* ... */` | Multi-line comment |
